# MemeScraper

MemeScraper is a web scraper for [CoronavirusMeme](https://old.reddit.com/r/Coronavirus_Meme/) that returns a random meme among the first 10.

This mini-project is part of a bigger project for [HackQuarantine](https://hackquarantine.com/).

## Prerequisites
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install BeautifulSoup4.

```bash
pip install BeutifulSoup4
```

## Usage
Open your terminal in the folder where the scirpt is and run the scirpt inthe following way:

```bash
python scraper.py
```
